# Restaurant Management System

A console-based **Restaurant Management System** written in C++ that demonstrates key Object-Oriented Programming (OOP) design patterns. 

## 🚀 Features
* **Customer Interface:** Real-time table reservations, item ordering, automated billing, and file-based feedback[cite: 1].
* **Manager Interface:** Protected by credentials (`admin123`) to track revenue analytics, view active table statuses, and review customer feedback[cite: 1].

## 🛠️ Concepts Demonstrated
* **OOP Paradigms:** Encapsulation, Abstraction, and Class structures[cite: 1].
* **Polymorphism & Inheritance:** Employs runtime polymorphism using an abstract base class (`User`) to toggle between user roles cleanly[cite: 1].
* **File Handling (`<fstream>`):** Persistent data logging for order records (`orders.txt`) and customer input (`feedback.txt`)[cite: 1].

## 📂 File Directory Structure
* `RMS FINAL.cpp` — Main program source file[cite: 1].
* `orders.txt` — Autogenerated client order receipts[cite: 1].
* `feedback.txt` — Persistent user reviews[cite: 1].
